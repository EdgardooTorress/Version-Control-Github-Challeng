3.  What are the steps to fork a GitHub repository?

- Open github and look for the repository you want to fork, and open it.
- Select where you want to fork the repository.
- Wait for the forking process to complete. Once the forking process is done, you'll have your copy of the repository under your GitHub account.
- You can now clone this forked repository to your local machine (terminal using git tool) and start working on it.
